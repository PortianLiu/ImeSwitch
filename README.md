# ImeSwitch - 输入法切换器

**为多输入法用户提供便利，支持便捷的输入法切换操作**

起因是主包自己用的输入法是自定义双拼，但是也很喜欢某包的语音输入，可是它的双拼尚不完善，而且还未必愿意支持自定义双拼，所以无法完全使用其中一方，于是开发此项目。

## 功能特性

### 快捷开关操作
- **短按快捷开关**：在已启用的输入法之间循环切换
- **长按快捷开关**：展开输入法选择列表，点击直接切换到指定输入法

### 输入法显示
- 自动获取并显示输入法的真实名称
- 支持系统输入法和第三方输入法
- 针对Android 11+包可见性限制进行优化

## 系统要求

- Android 7.0 (API 24) 及以上
- 针对Hyper OS 3 (Android 15) 进行测试

## 如何使用

1. 使用==Root权限==或者连接电脑后使用==Adb权限==根据提示进行授权
3. 下拉控制中心，进入快捷面板设置，添加"输入法切换"磁贴

## 可选配置

如果输入法名称显示不正常，可以尝试授予"查询所有应用"权限，有可能能够改善。

## 构建项目

```bash
# 克隆项目
git clone https://github.com/PortianLiu/ImeSwitch.git

# 进入项目目录
cd ImeSwitch

# 构建项目
gradle build

# 安装到设备
gradle installDebug
```
